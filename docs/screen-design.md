# 画面設計

## 画面一覧

Learning Note v1では、以下の画面を作成する。

1. ホーム画面
2. メモ作成・編集画面
3. メモ詳細画面
4. 分類管理画面

---

## 1. ホーム画面

### 目的

学習メモを検索・確認し、新規メモを追加できるようにする。

### 表示内容

- アプリ名
- 検索欄
- 最近のメモ一覧
- 分類一覧
- メモ一覧
- 新規追加ボタン

### 操作

- 検索欄に入力すると、メモ一覧を絞り込む
- 分類を選択すると、その分類のメモを表示する
- メモを選択すると、メモ詳細画面へ移動する
- 追加ボタンを押すと、メモ作成画面へ移動する
- 分類管理ボタンから分類管理画面へ移動する
