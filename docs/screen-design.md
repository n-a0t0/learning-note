# 画面設計

## 画面一覧

Learning Note v1では以下の画面を作成する。

1. ホーム画面
2. メモ作成・編集画面
3. メモ詳細画面
4. 分類管理画面

---

# 1. ホーム画面

## 目的

学習メモを検索・確認し、新規メモを追加できるようにする。

## 表示内容

- アプリ名
- 検索欄
- 最近のメモ一覧
- 分類一覧
- メモ一覧
- 新規追加ボタン

## 操作

- 検索欄へ入力するとメモを検索する
- 分類を選択すると該当するメモ一覧を表示する
- メモをクリックするとメモ詳細画面へ移動する
- 「＋」ボタンを押すとメモ作成画面へ移動する
- 分類管理ボタンを押すと分類管理画面へ移動する

## 画面イメージ

    ┌────────────────────────────┐
    │ Learning Note              │
    │ [検索欄                  ] │
    ├────────────────────────────┤
    │ 最近のメモ                 │
    │ ・GitHub README            │
    │ ・C# クラス                │
    ├────────────────────────────┤
    │ 分類                       │
    │ [C#] [Git] [SQL] [Java]    │
    ├────────────────────────────┤
    │ メモ一覧                   │
    │ ・タイトル / 分類 / 理解度 │
    │ ・タイトル / 分類 / 理解度 │
    │                        ＋  │
    └────────────────────────────┘

# 3. メモ詳細画面

## 目的

登録した学習メモを閲覧する。

## 表示内容

- タイトル
- 分類
- 理解度
- 学習内容
- メモ
- 作成日
- 更新日

## 操作

- 編集
- 削除
- 戻る

## 画面イメージ

    ┌────────────────────────────┐

    │ 分類：Git                  │
    │ 理解度：3                  │
    │ 作成日：2026/07/02         │

    │ 学習内容                   │
    │ ...                        │
    ├────────────────────────────┤
    │ メモ                       │
    │ ...                        │
    │                            │
    │ [編集] [削除] [戻る]       │

# 4. 分類管理画面

## 目的

分類を追加・編集・削除する。

## 表示内容

- 分類一覧
- 分類入力欄
- 追加ボタン
- 編集ボタン
- 削除ボタン

## 操作

- 分類を追加する
- 分類を編集する
- 分類を削除する

## 画面イメージ

    ┌────────────────────────────┐
