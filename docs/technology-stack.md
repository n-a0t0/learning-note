
# 技術選定

## 概要

Learning Note v1では、C#を使用したWindowsデスクトップアプリとして開発する。

---

## 使用技術

| 項目 | 技術 |
|---|---|
| 言語 | C# |
| UI | WPF |
| フレームワーク | .NET 8 |
| DB | SQLite |
| ORM | Entity Framework Core |
| IDE | Visual Studio 2022 |
| バージョン管理 | Git / GitHub |

---

## 技術選定理由

### C#

過去の開発経験があり、今後の転職でも活かしやすいため。

### WPF

Windows向けのデスクトップアプリを作成でき、C#の経験を活かしやすいため。

### SQLite

ローカル環境で簡単に利用でき、学習ノートのような個人向けアプリに適しているため。

### Entity Framework Core

C#のクラスとデータベースを結びつけて管理でき、実務でも使われる技術であるため。

### Git / GitHub

開発履歴や設計資料を管理し、ポートフォリオとして公開するため。
