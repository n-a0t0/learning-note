# 画面遷移

## 画面一覧

- ホーム画面
- メモ作成・編集画面
- メモ詳細画面
- 分類管理画面

---

## 画面遷移図

```text
ホーム画面
│
├── ＋ボタン
│      ↓
│   メモ作成画面
│      │
│      ├── 保存
│      │      ↓
│      │   ホーム画面
│      │
│      └── キャンセル
│             ↓
│         ホーム画面
│
├── メモ選択
│      ↓
│   メモ詳細画面
│      │
│      ├── 編集
│      │      ↓
│      │   メモ編集画面
│      │      │
│      │      └── 保存
│      │             ↓
│      │        メモ詳細画面
│      │
│      ├── 削除
│      │      ↓
│      │   ホーム画面
│      │
│      └── 戻る
│             ↓
│         ホーム画面
│
└── 分類管理
       ↓
   分類管理画面
       │
       ├── 追加
       ├── 編集
       ├── 削除
       │
       └── 戻る
              ↓
          ホーム画面
```

---

## 各画面の役割

### ホーム画面

アプリの入口。
メモを探す・追加する。

---

### メモ作成・編集画面

学習内容を登録・編集する。

---

### メモ詳細画面

登録した内容を閲覧する。

---

### 分類管理画面

分類を管理する。
